## 📝 클라우드 컨설팅 신청

전문 컨설턴트와의 상담을 거쳐 진행됩니다.

**\*표시**는 **필수 입력 내용**입니다.

-----

### 👤 사업자 정보

| 항목 | 입력 내용 | 비고 |
| :--- | :--- | :--- |
| **사업자 유형\*** | ⚪ 개인 / ⚪ 개인사업 / ⚪ 법인사업 | 필수 선택 |
| **사업자등록번호\*** | `[사업자등록번호 입력]` | 필수 입력 |
| **회사명\*** | `[회사명 입력]` | 필수 입력 |

-----

### 📞 담당자 정보 및 연락처

| 항목 | 입력 내용 | 비고 |
| :--- | :--- | :--- |
| **담당자명\*** | `[담당자명 입력]` | 필수 입력 |
| **휴대 전화번호\*** | `[선택박스: 010, 011, 017 등]` - `[4자리 입력]` - `[4자리 입력]` | 필수 입력 |
| **일반 전화번호** | `[선택박스: 02, 031, 070 등]` - `[3~4자리 입력]` - `[4자리 입력]` | 선택 사항 |
| **E-mail\*** | `[E-mail 주소 입력]` | 필수 입력 |
| **소재지\*** | `[소재지 입력]` | 필수 입력 |

-----

### 💡 컨설팅 신청 내용

| 항목 | 입력 내용 | 비고 |
| :--- | :--- | :--- |
| **컨설팅 신청 내용\*** | `[텍스트 영역: 컨설팅을 원하는 내용(예: 마이그레이션, 인프라 구축 등)을 상세히 입력]` | 필수 입력 |
| **신청경로\*** | `[선택박스: 영업직원, 웨비나, 뉴스레터, e-DM, 세미나/박람회, 언론기사, 유튜브, 블로그, AICANSMILE 포털, 기타]` | 필수 선택 |

-----

### ✅ 개인정보 수집/이용 동의 (필수)

| 항목 | 내용 |
| :--- | :--- |
| **수집 항목** | 신청인의 업체명, 담당자명, 휴대전화번호, 일반전화번호, e-mail 주소, 소재지, 사업자등록번호 |
| **수집 목적** | 컨설팅 요청 응대를 위한 신청자 정보확인 및 요청 내용 확인 |
| **수집 기간** | 가. 이용기간 : 컨설팅 진행 기간<br>나. 보유기간 : 컨설팅 요청 이후 6개월 간 |

> ※ 위의 개인정보 수집·이용에 대한 동의를 거부할 권리가 있습니다. 그러나 동의를 거부할 경우 원활한 서비스 제공에 일부 제한을 받을 수 있습니다.
